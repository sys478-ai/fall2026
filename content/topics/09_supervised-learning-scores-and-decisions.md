## Topic / Focus

This topic introduces the basic machine learning pipeline: training data, model fitting, inference, and output. The emphasis is not on advanced mathematics. It is on helping students see what machine learning is actually doing when a system is said to learn from examples.

Learning in this technical sense means detecting patterns in data and adjusting internal parameters so future inputs can be classified, scored, or predicted. That is powerful, but it is also narrower than the richer forms of human learning discussed in Module 1.

By the end of this topic, you should be able to describe how a machine learning system moves from data to model to output, and why that process should not be confused with learning in a fuller human sense.

## Guiding Questions

- What does training a model actually mean?
- What is the difference between training and inference?
- What kinds of patterns are models learning from data?
- In what sense is this called learning, and in what sense is it not?
- Why does the metaphor of learning both help and mislead?
